# Clúster IALab — Documentación Oficial CENIA

Bienvenido al repositorio oficial de documentación del **Clúster IALab de CENIA**.

Este repositorio centraliza la información general, guías de uso, buenas prácticas y recursos asociados al uso del clúster para investigación, desarrollo y ejecución de cargas de trabajo de inteligencia artificial y cómputo de alto rendimiento.

---

## 📚 Wiki Oficial

La documentación completa y actualizada se encuentra disponible en la Wiki del repositorio:

➡️ **https://github.com/Sysadmins-CENIA/Cluster-IALab-Docs/wiki**

La Wiki incluye información detallada sobre:

- Políticas de Solicitud y Asignación de recursos
- Inducción al Clúster
- Almacenamiento
- Uso de SLURM
- Chequeo de eficiencia Slurm con "sact y "seff"
- Ejemplos de SLURM Scripts
- Uso de SSH
- FAQ ‐ Preguntas Frecuentes
- Política de Uso

---

## 🚀 Objetivo del repositorio

Este repositorio busca:

- Centralizar la documentación técnica del clúster
- Facilitar el onboarding de nuevos usuarios
- Estandarizar flujos de trabajo
- Compartir ejemplos y configuraciones recomendadas
- Mantener una referencia viva y colaborativa
